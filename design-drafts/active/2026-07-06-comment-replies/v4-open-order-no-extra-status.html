Comment handling protocol

?Codex reads comments in document order and only resolves a comment after it has clearly satisfied the user's request.

✓Clear comments

When the request is clear, Codex completes the work, replies on the comment, and marks it resolved.

?Ambiguous comments

When the request is unclear, Codex replies with a question and leaves the comment open in the latest version.
