# Mermaid 大图查看器设计

## 1. 背景与目标

当前 `MermaidBlock` 将生成的 SVG 限制为容器宽度。宽流程图和长时序图会整体缩小，节点与连线文字难以辨认。容器只提供横向滚动，缺少放大和二维移动能力。

本次改动为 Mermaid 图表增加独立的大图查看器。正文保持适应宽度的静态预览，用户通过图表右上角的“放大查看”按钮进入浮层，在浮层内缩放和移动图表。

## 2. 范围

### 2.1 包含范围

- 所有成功渲染的 Mermaid 图表显示“放大查看”入口。
- 浮层覆盖浏览器可视区域，背景使用低透明度遮罩。
- 浮层提供缩小、比例、放大、适应窗口和关闭控件。
- 支持鼠标拖动、触控板双指平移和捏合缩放。
- 支持 `Escape` 关闭和双击放大一级。
- 保持浅色、深色主题一致的层级与密度。

### 2.2 非目标

- 不修改 Mermaid 源码或渲染主题。
- 不保存不同图表的缩放和位置状态。
- 不提供导出、下载、全屏 API 或图表编辑能力。
- 不在正文预览中拦截滚轮或触控板手势。

## 3. 方案选择

评估过三种入口：图内文字按钮、整图点击和极简展开图标。最终采用图内文字按钮。

该入口在首次使用时可发现性较高，也能明确表达动作。按钮在图表右上角显示，样式沿用紧凑次级按钮。整图本身不承担点击语义，避免与文本选择和页面滚动产生冲突。

## 4. 组件结构

`MermaidBlock` 继续负责 Mermaid 初始化、SVG 生成和错误态。成功生成 SVG 后，它渲染正文预览和“放大查看”入口。

新增独立的图表查看器组件，职责包括：

- 使用原始 SVG 内容绘制浮层画布。
- 管理打开状态、缩放比例和二维位移。
- 处理按钮、键盘、鼠标、滚轮和触控板事件。
- 根据查看器尺寸和 SVG 原始尺寸计算适应窗口比例。
- 打开时保存原焦点并聚焦关闭按钮，关闭后恢复焦点。

查看器通过 React portal 挂载到 `document.body`，避免被 Markdown 容器的 `overflow` 或层叠上下文裁剪。

## 5. 交互规则

### 5.1 打开与关闭

- 点击“放大查看”打开浮层。
- 初始状态执行“适应窗口”，图表完整显示在可用画布内。
- 点击关闭按钮、遮罩空白区域或按 `Escape` 关闭。
- 点击画布和工具栏不会触发遮罩关闭。

### 5.2 缩放

- 比例范围为 `25%` 至 `400%`。
- 工具栏的加减按钮按固定步长调整比例。
- 触控板捏合或 `Ctrl/Cmd + 滚轮` 以指针位置为中心缩放。
- 双击图表以指针位置为中心放大一级。
- 比例控件显示取整后的当前百分比，点击比例执行“适应窗口”。

### 5.3 移动

- 鼠标按下并拖动图表可移动画布。
- 触控板双指滑动在浮层内移动画布。
- 普通滚轮增量用于平移；组合缩放手势不再触发平移。
- 拖动期间光标从 `grab` 切换为 `grabbing`。

### 5.4 页面隔离

- 浮层打开时锁定页面背景滚动。
- 浮层关闭后恢复原有滚动状态。
- 正文 Mermaid 图不注册滚轮缩放行为，文档阅读与滚动保持现状。

## 6. 视觉与无障碍

- 遮罩层覆盖当前视口，图表画布使用现有面板背景、12px 圆角和定义明确的边框。
- 工具栏位于画布右上角，使用 26px 至 30px 的紧凑控件和现有图标尺度。
- 图表查看器使用对话框语义，提供可读名称。
- 所有按钮提供中文无障碍名称、可见焦点和禁用状态。
- 缩放到上下限时禁用对应按钮。
- `prefers-reduced-motion: reduce` 下关闭非必要过渡。

## 7. 边界与错误处理

- Mermaid 渲染失败时保持现有错误块，不显示查看入口。
- SVG 缺少有效尺寸时使用 `viewBox` 计算；仍无法取得尺寸时按当前容器尺寸执行适应窗口。
- 查看器尺寸变化时重新计算可用范围，但不在用户已经手动缩放后强制重置视图。
- 组件卸载时移除全局事件并恢复页面滚动状态。

## 8. 验证计划

组件测试覆盖以下行为：

- Mermaid 成功渲染后出现“放大查看”入口。
- 打开查看器后显示对话框和工具栏，初始执行适应窗口。
- 加减按钮更新比例并遵守上下限。
- `Escape` 和关闭按钮关闭浮层并恢复入口焦点。
- 指针拖动更新位移。
- 组合滚轮缩放与普通滚轮平移走不同路径。
- 错误态不显示查看入口。

项目级验证包括 `pnpm test`、`pnpm lint`、`pnpm fmt:check` 和 `pnpm build`。运行时在目标 Markdown 页面检查浅色与深色主题、侧栏展开与收起、查看器打开与关闭，以及长时序图的缩放和移动。
